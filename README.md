# 🤖 MADMAX AI Chatbot

A modern AI chatbot web app built using **Flask + Gemini API + Tailwind CSS** with a Kick Buttowski anime themed UI.

## 🚀 Features

* Dark anime themed chat interface
* Kick Buttowski styled design & avatars
* Flask backend API
* Google Gemini AI integration
* Real-time chat responses
* Typing animation & smooth UI

## 🛠️ Tech Stack

* Frontend: HTML, Tailwind CSS, JavaScript
* Backend: Python Flask
* AI Model: Google Gemini API

## 📂 Project Structure

```
app.py / app2.py   -> Flask backend server
index.html         -> Chat UI (frontend)
README.md          -> Project documentation
```

## ▶️ How to Run

### 1️⃣ Install requirements

```bash
pip install flask flask-cors google-generativeai
```

### 2️⃣ Run Flask server

```bash
python app2.py
```

### 3️⃣ Open in browser

```
http://127.0.0.1:5000
```

## 💬 How it Works

* User sends message from chat UI
* Flask receives message through `/chat` API
* Gemini AI processes the message
* Response is returned and displayed in chat

Kick Buttowski themed chat interface with AI response system.
